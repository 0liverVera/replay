# $REPLAY

> Hit Replay. Run It Back.

**$REPLAY** is a retro arcade-themed memecoin on Solana with a playable arcade game. Fair launch. No presale. No insiders. Just a dev who built something you can actually play.

---

## The Narrative

Crypto used to be fun. Fair launches. Real communities. Builders who actually built.

Then the suits showed up. VCs. Insiders. KOL rounds. The game got rigged.

$REPLAY is the reset button. One coin. One game. One community. No presale. No insiders. Just quarters.

---

## The Game

Endless runner / obstacle dodger built with HTML5 Canvas.

- **Move** left/right with arrow keys, A/D, or tap left/right on mobile
- **Dodge** falling obstacles: VCs, insiders, KOLs, briefcases, rugpulls
- **Collect** quarters (¢) for bonus points
- **RESET** power-up clears all obstacles and grants brief invincibility
- Speed ramps up over time
- High score saved to localStorage
